# devops identity show

resolves the identity for a principalName

```
azex devops identity show --tenant ... --organization ... --principalName ...
```
```JSON
{
  "tenant"            : "...",
  "organization"      : "...",
  "principalName"     : "...",
  "subjectDescriptor" : "...",
  "identityDescriptor": "..."
}
```
